# 📜 The Scribe's Scroll: Wordle Edition

Welcome to **The Scribe's Scroll**, a vintage-themed word puzzle game and quest log. This application combines the classic "Wordle" mechanics with a rich, historical atmosphere.

## 🏺 Aesthetic Design
- **Parchment Interface**: A textured scroll overlay on an antique cream background.
- **Wooden Desk Environment**: Set against a dark, candle-lit wooden surface.
- **Historical Typography**: Featuring serif and typewriter fonts for an authentic scribe feel.
- **NO PURPLE**: Strictly enforced color palette of sepia, gold, burnt umber, and charcoal ink.

## 🎮 Game Rules (Wordle System)
Instead of guessing individual letters, you must decipher entire **Scripts** (Words).
- **Gold Tiles/Keys**: The letter is in the correct position.
- **Sepia Tiles/Keys**: The letter exists in the script but is in the wrong position.
- **Ink-Faded Tiles/Keys**: The letter does not exist in the script.
- **Virtual Keyboard**: An interactive keyboard reflects your progress, showing which letters have been cast and their accuracy.
- **Attempts**: You have 6 attempts to solve the puzzle.
- **Difficulties**:
  - **Novice**: 4-letter words.
  - **Scholar**: 5-letter words.
  - **Sage**: 6+ letter words.

## 🖋️ Scribe Meta Features
- **XP & Ranks**: Earn Experience Points (XP) by winning games and completing quests. Advance through ranks from *Initiate Scribe* to *Grand Archivist*.
- **Quest Log**: A built-in To-Do list. Completing daily quests grants XP. Winning the game automatically checks off the "Slay the Word Dragon" quest.
- **Ancient Wisdom**: Click the **Ink Bottle** decoration to reveal a Wisdom Seal containing proverbs from master scribes.
- **Hints**: If stuck, you can "Seek a Hint" for an XP cost.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Layout, unrolling animations, and state transitions.
- **Vanilla JavaScript**: Game logic, persistence (localStorage), and interactive meta-features.

## 🚀 Installation & Usage
1. Open `index.html` in any modern web browser.
2. Choose your difficulty and type your first guess.
3. Switch between the **Word Guessr** and **Quest Log** using the tabs on the scroll.
